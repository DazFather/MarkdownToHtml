# MarkdownToHtml
A Function that allow you to transform a Markdown text into HTML
> Tiny utility that I needed to create myself because I coudn't find another ready to use library.


## Supported tags:
| Markdown | HTML           |
| -------- | -------------- |
| **       | &lt;strong&gt; |
| __       | &lt;em&gt;     |
| *        | &lt;b&gt;      |
| _        | &lt;i&gt;      |
| #        | &lt;h1&gt;     |
| ##       | &lt;h2&gt;     |
